## 🎭 Animations et Interactions

### Hover sur Carte
```
État normal:
┌──────────────┐
│   🍞         │  ← Bordure grise
│              │     Position: Y=0
└──────────────┘

Au survol (hover):
╔══════════════╗
║   🍞↻        ║  ← Bordure colorée en haut
║              ║     Position: Y=-8px (élevée)
║              ║     Icône: rotation(5deg) + scale(1.1)
╚══════════════╝     Ombre portée augmentée
```

### Bouton Commander
```
État normal:
[ 🛒 Commander ]  ← Fond primaire

Hover:
[ 🛒 Commander ]  ← Scale(1.05)
                     Ombre portée

Clic:
[ 🛒 Commander ]  ← Scale(0.98)
                     Feedback tactile
```

## 📊 Hiérarchie Visuelle

```
1. Icône du produit (4rem) ──────────► Attention immédiate
2. Badge "Vedette" (si applicable) ──► Accent visuel
3. Titre du produit (1.25rem) ──────► Identification
4. Description (0.95rem) ────────────► Contexte
5. Prix (1.75rem, couleur) ──────────► Information clé
6. Détails (poids, allergènes) ──────► Informations secondaires
7. Bouton Commander ─────────────────► Call-to-action
```
